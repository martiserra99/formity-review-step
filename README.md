# Formity — Review Step

Starting point for the [Review step](https://formity.app/docs/guides/review-step) guide on the Formity docs.

## Overview

This repository contains a multi-step job application form built with Formity and React Hook Form. It has four steps — personal details, background, preferences, and about you — followed by a review step that summarises the collected data. The edit buttons on the review step are not wired up yet. The guide walks you through making them work.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-review-step
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
